En este proyecto he creado una base de datos sobre **relojes** con distintos campos y valores
incluye 3 colecciones de datos adicionales generadas a partir del insert original

Hay 3 carpetas:

-En la carpeta **doc** hay un pdf en el que se explican la base de datos, las colecciones y el uso de MongoDB Atlas

-En la carpeta **src** estan los scripts utilizados para crear la base de datos junto con algunas consultas

-En la carpeta **datos** se encuentran los archivos json resultado de utilizar mongoexport
